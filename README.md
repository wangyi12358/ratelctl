# Ratelctl
Ratelctl 是一款能够快速初始化项目的命令行工具，它能够帮助你快速初始化项目，让你专注于业务开发。  
并且会集成通用的工具，比如：代码格式化、代码检查、单元测试、代码覆盖率、构建、发布等。

## 安装
```shell
$ go install github.com/wangyi12358/ratelctl
```

## 使用
使用 ratelctl 来初始化你的项目
```shell
$ ratelctl init
```

## 支持的模版项目

- React project
- React admin project
- Nestjs project
- Go Gin project
- Go-zero project
- Nodejs Cli project
