# Ratelctl
Ratelctl 是一款能够快速初始化项目的命令行工具，它能够帮助你快速初始化项目，让你专注于业务开发。  
并且会集成常用工具、配置、业务逻辑，让你的项目更加完善。

## 安装
```shell
$ go install github.com/wangyi12358/ratelctl@latest
```

## 使用
使用 ratelctl 来初始化你的项目
```shell
$ ratelctl init
```

## 支持的模版项目

- React project
- React admin project
- Nestjs project
- Go Gin project
- Go-zero project
- Nodejs Cli project
